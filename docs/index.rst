Wiki Documentation
==================

``wiki`` is a command-line tool for maintaining a wiki as a folder tree of
markdown pages indexed at every level by generated ``_index.md`` files. You
write the pages; the tool keeps the index links, frontmatter, and headings in
sync with the filesystem. It is built for developers and coding agents who
keep project knowledge in git alongside the code.
